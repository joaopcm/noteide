To start using Snipshare, follow these simple steps:

<Steps>
  <Step title="Sign up for a Snipshare account">
    You can create your account either by signing up with your email address and a password or by using your GitHub account.

    <Card title="Create my account" icon="rocket" href="https://app.snipshare.co" target="_blank">
      Click here to create your account
    </Card>
  </Step>

  <Step title="Create your first organization">
    An organization is a group of team members who can collaborate on code snippets. You can create multiple organizations to manage different projects or teams.
  </Step>

  <Step title="Invite team members to join your organization">
    You can invite your team members to join your organization by sending them an invitation link. Once they accept the invitation, they can start collaborating on code snippets with you.
  </Step>

  <Step title="Create projects and start adding code snippets">
    You can create projects to organize your code snippets. Each project can contain multiple code snippets that you can edit, run, and share with your team members.
  </Step>

  <Step title="Collaborate, share, and run your code snippets effortlessly">
    Snipshare provides a powerful code editor that supports syntax highlighting, code completion, and code execution. You can collaborate with your team members, share code snippets with external users, and run your code snippets to see the output.
  </Step>
</Steps>

Explore the power of Snipshare and revolutionize the way you manage and collaborate on code snippets. Happy coding!
